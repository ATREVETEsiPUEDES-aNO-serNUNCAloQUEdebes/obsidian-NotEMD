# Notemd v1.8.8

## Aspectos destacados

- **Sidebar API La observabilidad esta lista para su uso practico.**：Rapido deep-debug Cambiar、retry-aware Pruebe la semantica y la compacidad en vivo. API activity La historia ahora ha aterrizado junta mientras continua asegurando `Log output` Visibles。
- **`Vista previa de graficos` Ahora es la entrada real al producto guardado.**：apoyado Mermaid、JSON Canvas、Vega-Lite con HTML Los archivos graficos se pueden obtener una vista previa directamente y no es necesario volver a generar la salida guardada.。
- **La liberacion del valor real se completa simultaneamente.**：Metadatos de la version del paquete y resumen de la ventana emergente de bienvenida de las dos ultimas actualizaciones.、README Marcacion de versiones y bilinguismo release notes Alineado con `1.8.8`。

## Nuevas funciones

- Se agrego soporte de vista previa directa para las siguientes fuentes de graficos guardados.：
  - Mermaid Markdown fence
  - Vega-Lite Markdown fence
  - Vega-Lite JSON Archivos fuente（`.json`）
  - JSON Canvas Documentos（`.canvas`）
  - HTML Documentacion del producto（`.html`）
- Extension canonical preview Comandos y sidebar Portal: los archivos de productos guardados admitidos ahora pueden activar directamente vistas previas, que ya no se limitan a Markdown Archivo actual。
- en sidebar `Log output` footer header Agregar acceso directo en `Deep debug` controlar。
- nuevo request-scoped API Estado de la actividad de la prueba, cobertura. waiting、accepted、receiving、Salud de tareas largas、received con interrupted Etapa。
- Anadido compacto active/recent API activity Bloquear, apoyar inline per-request history Exportar con informes。

## Reparacion y Robustez

- Vista previa guardada Mermaid Ya no se vuelve a ingresar el enlace generado al dibujar, por lo que se elimina direct preview fallback En el proceso `requested intent` mismatch Fracaso。
- legacy Mermaid Ya no se ejecuta `erDiagram` Tal compatibilidad Mermaid Explicito requested intent Repliegue forzado a `mindmap`。
- API activity El historial ahora tiene limite superior y capacidades de plegado para evitar ocupar el area de salida del registro en multiples escenarios de solicitud.。
- Preview tooltip Copiar ahora se alinea con el comportamiento real: indica explicitamente que obtiene una vista previa de las fuentes de graficos compatibles o los archivos de productos guardados en lugar de "genera pero no guarda".”。
- request-scoped La semantica en vivo de las pruebas ahora mantiene una coherencia mas estable en escenarios de reintento y concurrencia.。
